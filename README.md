# PathLab - Pathology Management SaaS

A comprehensive, production-ready Pathology Management System built with Next.js 14, Firebase, and Tailwind CSS.

## Features

- **Dashboard** - Real-time analytics with charts and stats
- **Patient Management** - CRUD operations with search and filtering
- **Sample Tracking** - End-to-end sample lifecycle management
- **Report Generation** - PDF generation with QR codes and auto-flagging
- **Billing System** - Invoicing with discounts, tax, payment tracking
- **Expenses** - Track lab operational costs
- **Investigations** - Manage test orders and results
- **WhatsApp/SMS Integration** - Send reports and notifications
- **Letterhead Designer** - Customizable report templates
- **Subscription System** - Premium plans with UPI payment
- **B2B Module** - Doctor referrals and corporate accounts
- **Admin Panel** - Super admin controls, payment approvals
- **Analytics** - Revenue, patient, and test distribution charts

## Tech Stack

- **Frontend:** Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS
- **Backend:** Firebase (Auth, Firestore, Storage, Analytics)
- **UI:** Framer Motion, Recharts, React Hot Toast, React Icons
- **PDF:** jsPDF, jsPDF-AutoTable, html2canvas
- **QR:** qrcode library
